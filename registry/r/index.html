taskflow registry

Built-in registry for @taskflow-corp/cli (shadcn-style harness distribution for multi-agent pipelines).

install a harness

npx @taskflow-corp/cli@latest add <name>

endpoints

  • /registry.json — this namespace's item list
  • /registries.json — master index of public namespaces
  • /<name>.json — a single registry item (e.g. /example-hello.json)
